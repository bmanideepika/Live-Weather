**Live Weather App**

A real-time weather reporting web application that fetches live weather data using the OpenWeatherMap API.
The application allows users to search for any city and instantly view the current weather conditions.

This project demonstrates API integration, asynchronous JavaScript, and responsive UI design using modern frontend technologies.

**Features**

- Search weather by **city name**

- Displays **temperature, humidity, wind speed**

- Shows **weather conditions and icons**

- Real-time data using **OpenWeatherMap API**

- Fully **responsive design**

- Clean UI with **Google Fonts and Bootstrap**

**Technologies Used**

-HTML5 – Page structure

-CSS3 – Styling and layout

-Bootstrap – Responsive design

-JavaScript (ES6) – Application logic

-jQuery – DOM manipulation

-OpenWeatherMap API – Weather data
